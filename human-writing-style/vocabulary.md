# AI vocabulary — full list with diagnostics

Sibling reference file for the `human-writing-style` skill. This expands rule 4 in `rules.md`.

**Load this file when** editing or rewriting existing text (where word-level swaps matter), or when you suspect a vocabulary cluster and want to confirm which words count.

---

## Calibration

These words became dramatically overused in LLM output after 2022. Single occurrences are usually fine. The tell is **clustering** — three or more within a few paragraphs.

- In a piece under 1,000 words: one is fine.
- In a longer piece: two or three spread apart is acceptable.
- Three or more within a few paragraphs: reads like AI wrote it. Fix.

**Exception:** if one of these words is the precise technical term for the domain (e.g., "foster" in developmental psychology, "intricate" in textile or watchmaking writing), use it freely. The problem is gratuitous usage, not the words themselves.

---

## The list

Each entry lists a replacement and, where useful, a diagnostic for deciding whether to cut or swap.

- *Additionally* (especially starting a sentence) → "Also", "And", or restructure. **Almost always removable without loss.**
- *delve* → "look into", "examine", "explore". **One of the strongest tells; humans rarely reach for this word.**
- *crucial* → "important", "necessary", or cut. **Usually filler — try deleting the sentence first, then the word, then substitute.**
- *enhance* → "improve", "strengthen", or be specific. **If it's doing real work, name *how* it improves.**
- *foster* → "encourage", "support", "build".
- *intricate / intricacies* → "complex", "complicated", "details".
- *interplay* → "relationship", "connection", "tension".
- *landscape* (abstract) → "field", "world", "situation", or cut. **Fine when literal; the tell is the abstract business-y usage.**
- *meticulous / meticulously* → "careful", "thorough", "precise". **Often overstates — ask whether "careful" is enough.**
- *pivotal* → "important", "major", "turning-point" (noun).
- *showcase* → "show", "display", "demonstrate".
- *tapestry* (abstract) → cut it entirely, almost always filler.
- *testament* → cut it; say what proves the point instead.
- *underscore* (verb) → "show", "make clear", "reinforce". **Mechanical metaphor; "show" almost always works.**
- *vibrant* → be specific about what makes it lively.
- *enduring* → "lasting", "long-running", or cut.
- *highlight* (verb) → "show", "point out", "note".
- *valuable* → say what the value actually is.
- *align with* → "match", "fit", "follow".

## How to apply

Don't mechanically search-and-replace. For each hit, walk this decision tree:

1. **Can the sentence go entirely?** Often the word is a symptom of a sentence that wasn't saying anything concrete. Delete it and see if the piece loses anything real.
2. **Can the word go?** Many of these are filler. "A crucial part of the plan" usually reads just as well as "a part of the plan" — or better, if you can say *which* part.
3. **Can you replace it with something more specific?** If you say "enhance customer experience," try naming *what* about the experience improved. That's the master rule doing its work.
4. **Only if none of the above apply, substitute.** A word swap is the weakest move and the last resort.
